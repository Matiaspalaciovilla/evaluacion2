Lenguajes que estamos aprendiendo

HTML (HyperText Markup Language) es el lenguaje estándar utilizado para crear y diseñar páginas web. Se compone de una serie de etiquetas que se utilizan para estructurar el contenido de una página web, como títulos, párrafos, imágenes, enlaces, formularios, entre otros elementos. HTML es un lenguaje de marcado que define la estructura y el significado del contenido de una página web, permitiendo que los navegadores web interpreten y muestren la información de manera adecuada para los usuarios.

CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado para controlar la presentación y el estilo visual de una página web creada con HTML. CSS permite definir el aspecto de los elementos HTML, como colores, fuentes, márgenes, tamaños, alineaciones, entre otros aspectos visuales. Al separar el contenido (HTML) de la presentación (CSS), se logra una mayor flexibilidad y control en la apariencia de un sitio web, facilitando la creación de diseños atractivos y consistentes. CSS se aplica a través de reglas que especifican cómo se deben visualizar los elementos HTML en un navegador web.

JavaScript

JS (JavaScript) es un lenguaje de programación utilizado principalmente en el desarrollo web para agregar interactividad y dinamismo a las páginas. A diferencia de HTML y CSS, que se centran en la estructura y el diseño visual de un sitio web, JavaScript se enfoca en la funcionalidad y la interacción del usuario. Con JavaScript, es posible realizar acciones como validar formularios, crear efectos visuales, manipular elementos de la página en tiempo real, realizar peticiones a servidores, entre otras muchas funcionalidades. JavaScript es un lenguaje versátil que se ejecuta en el lado del cliente, es decir, en el navegador del usuario, lo que permite crear experiencias interactivas y dinámicas en las páginas web.
